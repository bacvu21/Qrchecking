    This is the guide for using QR check code.
    With Vietnamese
    chương trình điều khiển arduino với tác vụ đóng ngắt role 


    Qrcheck.py là file tạo giao diện và mở file Qr_Checking
    file Qr_checking là file chính 
    sử dụng để quét QR và đọc dữ liệu của mã Qr 
    các file txt như save.txt và output1.txt là file lưu trữ dữ liệu của các file lần lượt là : 
    - save.txt sẽ lưu các đường link vừa mở của giao diện 
    - output1.txt sẽ lưu dữ liệu đối chiếu của mã Qrcode
